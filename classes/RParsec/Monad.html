  raise ArgumentError, 'monad cannot be nil' if m.nil?
  @monad = m;
  @this = v;

seq(other)

Run the seq operation on the encapsulated object following the monad law. If seq is not defined by the monad impl, use bind to implement.

[show source] 
# File lib/rparsec/monad.rb, line 39
def seq(other)
  if @monad.respond_to? :seq
    @monad.seq(other)
  else
    bind { |_x| other }
  end
end
